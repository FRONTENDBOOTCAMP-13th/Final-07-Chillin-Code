## 🌿 1. 프로젝트 소개 🌿

소피의 창고는 스튜디오 지브리 애니메이션 속 아름다운 순간들을<br/>
오래도록 간직할 수 있는 특별한 굿즈를 만나볼 수 있는 공간입니다.<br/>
지브리의 세계관이 담긴 피규어, 문구, 생활소품까지, 사랑스러운 아이템들을 한데 모아보았습니다.
